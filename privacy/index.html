١. مقدمة

نحن في إدارة تطبيق Sajjel AI (سجّل) نلتزم بحماية خصوصيتك. توضح هذه السياسة البيانات التي نجمعها وكيفية استخدامها وحمايتها.

٢. البيانات التي نجمعها

  • بيانات الأعمال التي تدخلها يدوياً أو صوتياً (أسماء العملاء، الموردين، المعاملات المالية، المنتجات).
  • بيانات المزامنة السحابية عبر Supabase (إذا فعّلت هذه الميزة).
  • بيانات استخدام مجهولة الهوية لتحسين الأداء (عدد الطلبات، الأخطاء التقنية).

٣. كيف نستخدم البيانات

  • تشغيل وظائف التطبيق: التعرف على الصوت، معالجة المعاملات، إصدار التقارير.
  • المزامنة السحابية بين أجهزتك.
  • تحسين دقة نماذج الذكاء الاصطناعي وأداء التطبيق.

٤. مشاركة البيانات مع أطراف ثالثة

نستخدم الخدمات التالية لتشغيل التطبيق:

  • Anthropic Claude API – معالجة الأوامر الصوتية والنصية بالذكاء الاصطناعي.
  • Supabase – التخزين السحابي والمزامنة.
  • Cloudflare Workers – بروكسي آمن لحماية مفاتيح API.

لا نبيع بياناتك لأي طرف ثالث ولأي غرض تجاري.

٥. تخزين البيانات وأمانها

  • بياناتك تُخزَّن على السحابة عبر Supabase لضمان إمكانية الوصول إليها من جهازك في أي وقت.
  • البيانات السحابية محمية بتشفير Supabase القياسي.
  • جميع الاتصالات مشفرة عبر HTTPS.

٦. حقوقك

  • يمكنك حذف جميع بياناتك من داخل التطبيق في أي وقت.
  • يمكنك طلب نسخة من بياناتك بمراسلتنا.

٧. التواصل معنا

لأي استفسار بخصوص الخصوصية: support@sajjelai.com

إدارة تطبيق Sajjel AI

آخر تحديث: يونيو ٢٠٢٦

1. Introduction

The Sajjel AI team is committed to protecting your privacy. This policy explains what data we collect, how we use it, and how we protect it.

2. Data We Collect

  • Business data you enter manually or via voice (customer names, suppliers, financial transactions, products).
  • Cloud sync data via Supabase (if you enable this feature).
  • Anonymous usage data to improve performance (request counts, technical errors).

3. How We Use Your Data

  • To operate app features: voice recognition, transaction processing, report generation.
  • To sync your data across your devices via the cloud.
  • To improve AI model accuracy and app performance.

4. Third-Party Data Sharing

We use the following services to operate the app:

  • Anthropic Claude API – AI processing of voice and text commands.
  • Supabase – Cloud storage and sync.
  • Cloudflare Workers – Secure proxy to protect API keys.

We do not sell your data to any third party for any commercial purpose.

5. Data Storage & Security

  • Your data is stored in the cloud via Supabase to ensure you can access it from your device at any time.
  • Cloud data is protected by Supabase's standard encryption.
  • All communications are encrypted via HTTPS.

6. Your Rights

  • You can delete all your data from within the app at any time.
  • You can request a copy of your data by contacting us.

7. Contact Us

For any privacy inquiries: support@sajjelai.com
